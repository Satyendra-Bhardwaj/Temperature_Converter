# Temperature_Converter
A Java console application that converts temperature between Celsius, Fahrenheit, and Kelvin.

## Features
- Celsius to Fahrenheit & Kelvin
- Fahrenheit to Celsius & Kelvin
- Kelvin to Celsius & Fahrenheit
- User-friendly menu
- Input validation
  
## Technologies Used
- Java
- Scanner Class
  
## How to Run
Compile: javac TemperatureConverter.java
Run: java TemperatureConverter
